LLM for translating learned speech in canines. 

Have you ever seen a talking Huskie on Youtube and wondered if that dog was really talking? Did it understand what it was saying? Could it answer direct questions? This project settles that debate with annalytical proof. 

This python based translation tool learns 7 specific words articulated by a dog.
This is only a 7-word model proof of concept, that can distinguish these 7 specific words, spoken by a dog, with up to 85% success rate. 
Included are 3 working versions of python script to run,

1.) Wav2Vec2 model -low accuracy (original design)
2.) Spectogram Model- high accuracy
3.) Spectrogram Model 1.1 - added weights for adjustment and more detailed output 

Contents of the zip file: 
1.) 30-50 audio samples of each of the 7 words, (spoken by a dog) 
2.) python codes, 
2.) Detailed notes about the words.

This is the small version (under 25mb) The original file is 249mb, but this is all you need to run the test.

This is released for educational purposes; I reatin all ownership of the dataset when it comes to commercial application.  

This data set is being released with the hopes someone will improve on the method. It also serves as the first overwhelming annalytical proof that dogs can speak in learned language.
