LLM for translating learned speech in canines. This is a 7-word model that can distinguish these 7 words, spoken by a dog, with up to 85% success rate. 
Included are 3 versions of the program, 
Wav2Vec2 model -low accuracy
Spectogram Model- high accuracy
Spectrogram Model 1.1 - added weights for adjustment and more detailed output 

The main contents are roughly 30-50 audio samples of each word, spoken by a dog, python codes, and notes about the words.
This is the small version (under 25mb) The original file is 249mb, but this is all you need to run the test.
